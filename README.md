# System Requirements

* Windows PowerShell 5.1 and .NET 4.6.1+
* PowerShell Core 6+

# Installing

To install globally:

```powershell
Install-Module -Name 'Carbon.Security'
Import-Module -Name 'Carbon.Security'
```

To install privately:

```powershell
Save-Module -Name 'Carbon.Security' -Path '.'
Import-Module -Name '.\Carbon.Security'
```
